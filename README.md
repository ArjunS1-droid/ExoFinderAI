# 🌍 ExoFinder AI - Exoplanet Transit Detection System

## 🚀 Project Description
This project is an AI-based pipeline that detects exoplanet transit signals from Kepler light curve data and evaluates habitability scores.

## 🧠 Features
- Light curve preprocessing
- Transit detection using ML
- Habitability index calculation
- Interactive Streamlit dashboard
- Visualization of exoplanet signals

## 📊 Workflow
Data → Preprocessing → Feature Extraction → ML Model → Habitability Score → Visualization

## 🛠️ Tech Stack
Python, Pandas, NumPy, Streamlit, Matplotlib

## 📂 Input Data
Kepler, TESS, K2 CSV / FITS files containing light curve data

## 📈 Output
- Detected exoplanets
- Transit graphs
- Habitability score dashboard
